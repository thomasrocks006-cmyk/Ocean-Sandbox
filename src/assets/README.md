# Assets Directory

Place your 3D models (GLB/GLTF), textures, and other assets here.

## Recommended Models
- Shark: Low-poly shark model (~5k polygons)
- Orca: Killer whale model
- Tuna: Schooling fish model
- Diver: Human diver with animations

## Texture Guidelines
- Use compressed formats (JPEG for color, PNG for alpha)
- Keep textures at reasonable sizes (1024x1024 or 2048x2048)
- Use texture atlases when possible

## HDRI Environments
- Place .hdr files here for custom environments
- Default preset "sunset" is used from @react-three/drei
